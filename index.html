HTML defines six levels of headings.A heading element implies all font changes,paragraph  breaks beforeand after,and any white space necessary to render the heading.

5.Difference between block and Inline.

Block:
Block Elements occupy the full width
Block elements always start in a new line and the browser automatically add some space before and after the element.
examples:div,p
Inline:
Inline elements occupy only sufficient width required.
Inline elements does not start in a new line.
examples: span,br
